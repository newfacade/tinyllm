# Claude Code 能力说明

Claude Code 是 Anthropic 官方的 Claude CLI 工具，可以帮助你完成各种软件工程任务。

## 主要能力

### 代码开发
- 编写新功能和实现需求
- 修复 bug 和错误
- 重构代码和优化性能
- 添加测试和验证功能

### 代码理解
- 阅读和分析现有代码
- 解释代码逻辑和工作原理
- 搜索代码库中的特定内容
- 理解项目结构和架构

### Git 操作
- 创建提交和推送更改
- 创建 Pull Request
- 查看提交历史和差异
- 管理分支

### 文件操作
- 读取、编辑、创建文件
- 搜索文件和内容
- 批量修改代码

### 开发工具
- 运行构建和测试命令
- 执行 shell 命令
- 安装依赖和管理包
- 调试和诊断问题

### 代码审查
- 检查代码质量
- 发现潜在问题和安全漏洞
- 提供改进建议

### 其他能力
- 支持多种编程语言
- 理解 Jupyter notebooks
- 查看和分析图片、PDF
- 网络搜索和获取最新信息

## 使用建议

- 给出清晰的任务描述
- 对于复杂任务，可以多轮对话
- 可以随时提问和寻求帮助
- 支持中英文交流

## 获取帮助

使用 `/help` 命令查看更多使用方法
